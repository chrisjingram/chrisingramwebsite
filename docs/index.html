  • Projects

MoveMore

I am the sole iOS developer for the mobile app supporting the MoveMore campaign in Sheffield providing scores, a breakdown of the users daily activity, and competitions. From concept to production I designed and developed the backend motion tracking technology, front end interface and scalable server architecture to capture large quantities of data from users of the app.

  • Learn more

Seeing With My Own Eyes

As a project for OAK Research Group, I developed a web app to enable visually impaired students to fully engage with the live demonstrations that take place in the modern classroom. Using WebRTC technology the teachers screen can be live streamed to the student's laptop with just one click, and without third party software.

Robert Ingram Decorators

Since 2006 I have designed, developed and maintained several versions of the website and portfolio for Robert Ingram Decorators, a painting and decorating company based in Leicestershire. The website was developed using HTML, CSS and Javascript.
